## 什么是 Spectra？

Spectra 是专为**国内开发者**打造的 Claude Code 客户端。它解决了国内开发者使用 Claude 时面临的三大核心痛点：网络不稳定、账号封禁风险、以及高昂的订阅成本。

通过 Spectra，你可以在国内网络环境下**直连** Claude Sonnet 4.6、Opus 4.6、Haiku 4.5 等全系列模型，无需任何代理工具，无需 API Key，无需信用卡，注册即用。
